# 推奨コマンド

## 開発
```bash
npm run dev       # 開発サーバー起動
npm run build     # ビルド (tsc + vite build)
npm run preview   # ビルドプレビュー
```

## 品質チェック
```bash
npm run lint      # ESLint
npm run typecheck # TypeScriptの型チェック
```

## 作業完了時
1. `npm run typecheck` - 型エラーチェック
2. `npm run lint` - Lintチェック
3. `npm run build` - ビルド確認
