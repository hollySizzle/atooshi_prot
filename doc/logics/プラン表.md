# Atooshi 手数料体系

## キャッシュポイント

| 項目 | 手数料率 | 発生タイミング | 備考 |
|:--|:--|:--|:--|
| **マッチング手数料** | 10% | コミッターへの報酬支払い時 | イノベーターが負担、コミッターの手取りは減らない |
| **CF手数料** | 15% | CF調達成功時 | 業界最安水準（CAMPFIRE 17%、Makuake 20%） |
| **シャドウストック手数料** | 10% | Bounty Pool設定時 | 実務遂行資金からの管理費 |

---

## 具体例

### Phase 0（Pre-Light）
```
コミッターへの報酬: 2万円
マッチング手数料: 2,000円（10%）
─────────────────────────
イノベーター支払総額: 2.2万円
コミッター手取り: 2万円（搾取排除）
Atooshi収益: 2,000円
```

### Phase 1（Light/CF）
```
CF調達額: 100万円
├─ CF手数料: 15万円（15%）
└─ 残り: 85万円
    ├─ シャドウストック（Bounty Pool）: 30万円
    │   └─ 手数料: 3万円（10%）
    └─ イノベーター手取り: 52万円

Atooshi収益: 18万円（15万 + 3万）
```

---

## 特徴

- **シンプルな料金体系**: 3つの手数料のみ
- **搾取排除**: コミッターの手取りは減らない
- **業界最安水準のCF手数料**: 15%
- **審査コストゼロ**: 可視化された情報で市場が判断
