# 02. プロジェクト一覧 (PJ-LIST)

## 基本情報

| 項目 | 内容 |
|------|------|
| 画面ID | PJ-LIST |
| URL | `/projects` |
| 主要ユーザー | 全員 |
| 目的 | プロジェクト探索、マーケットの広がりを示す |

---

## 投資家への訴求ポイント

- マーケットの多様性と広がり
- **既存CFとの違い**：実行体制が見える状態でのプロジェクト一覧
- Phase別の進行状況が一目で分かる

## プロジェクトカード詳細

### カード構成要素

```
┌─────────────────────────────────────────────────────────┐
│                    [カバー画像]                         │
│                      240px                              │
├─────────────────────────────────────────────────────────┤
│ [Phaseバッジ] [カテゴリバッジ]                           │
│                                                         │
│ プロジェクトタイトル（最大2行）                          │
│ 説明文（最大2行で省略...）                              │
│                                                         │
│ ┌─────────────────────────────────────────────────────┐│
│ │ 進捗バー                                   XX%      ││
│ │ ¥XXX,XXX / ¥X,XXX,XXX                              ││
│ └─────────────────────────────────────────────────────┘│
│                                                         │
│ ┌─────────────────────────────────────────────────────┐│
│ │ 実行体制: [●製造] [●デザイン] [○物流] [○CS]        ││
│ │ コミッター X名参画中                                 ││
│ └─────────────────────────────────────────────────────┘│
│                                                         │
│ 残りX日  支援者XX名                                     │
└─────────────────────────────────────────────────────────┘
```

### カード表示項目

| 項目 | 説明 | モックデータ例 |
|------|------|---------------|
| カバー画像 | プロジェクトのメイン画像 | placeholder画像 |
| Phaseバッジ | Pre-Light/Light/Middle | `Light` |
| カテゴリバッジ | プロジェクトカテゴリ | `ものづくり` |
| タイトル | プロジェクト名（最大40文字） | エコバッグ製造プロジェクト |
| 説明 | 概要（最大60文字で省略） | 廃棄素材を使った... |
| 進捗バー | 達成率を視覚化 | 75% |
| 調達金額 | 現在額/目標額 | ¥750,000 / ¥1,000,000 |
| 実行体制 | 役割の充足状況（●=確保/○=募集中） | 製造●, デザイン●, 物流○ |
| コミッター数 | 参画中のコミッター数 | 3名参画中 |
| 残り日数 | CF終了までの日数 | 残り12日 |
| 支援者数 | 支援者の累計 | 45名 |

### 実行体制バッジの凡例

| 状態 | 表示 | 色 |
|------|------|-----|
| 確保済み | ●（塗りつぶし） | `emerald-500` |
| 募集中 | ○（白抜き） | `gray-400` |
| 不要 | 表示なし | - |

## フィルター・ソート

### Phaseタブ

| タブ | 説明 |
|------|------|
| すべて | 全プロジェクト |
| Pre-Light | 準備中（実行証拠づくり） |
| Light | CF実施中 |
| Middle | テストマーケ中 |

### フィルター項目

| フィルター | 選択肢 |
|-----------|--------|
| カテゴリ | ものづくり、テクノロジー、食品、サービス、その他 |
| 調達状況 | 募集中、達成間近、達成済み |
| 必要スキル | エンジニア、デザイン、製造、物流、マーケ、CS |

### ソート

| ソート順 | 説明 |
|---------|------|
| 新着順（デフォルト） | 公開日が新しい順 |
| 達成率順 | 達成率が高い順 |
| 残り日数順 | 終了が近い順 |
| 支援者数順 | 支援者が多い順 |

## インタラクション

| 要素 | アクション | 遷移先/結果 |
|------|-----------|------------|
| 検索入力 | Enter/検索ボタン | フィルタリング実行 |
| Phaseタブ | クリック | 該当Phaseでフィルタ |
| フィルタードロップダウン | 選択 | フィルタリング実行 |
| ソートドロップダウン | 選択 | ソート実行 |
| プロジェクトカード | クリック | `/projects/:id` |
| もっと見るボタン | クリック | 追加読み込み |
