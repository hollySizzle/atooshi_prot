# 05. コミッターマッチング (CM-MATCH)

## 基本情報

| 項目 | 内容 |
|------|------|
| 画面ID | CM-MATCH |
| URL | `/match` |
| 主要ユーザー | コミッター |
| 目的 | スキルマッチするプロジェクト探索、参画申請 |

---

## 投資家への訴求ポイント

- 人材供給側の魅力・使いやすさ
- **Boarding Track**：単発→継続参画への昇格パス
- **スキルマッチング**：適切な人材配置の仕組み
- **セカンドキャリア人材**の活躍イメージ

## セクション詳細

### 1. プロフィールサイドバー

#### 基本情報

| 項目 | 表示内容 |
|------|---------|
| アバター | プロフィール画像 |
| 名前 | 表示名 |
| 肩書き | headline |
| Boarding Level | 現在のレベル + プログレスバー |

#### スキル一覧

| 表示項目 | 説明 |
|---------|------|
| スキル名 | 登録スキル |
| 習熟度バー | Beginner/Intermediate/Advanced/Expert |
| 編集ボタン | スキル編集モーダルを開く |

#### 希望条件

| 項目 | 表示内容 |
|------|---------|
| 週稼働時間 | XX〜XX時間 |
| 希望単価 | ¥X,XXX〜/h |
| Track希望 | Task / Boarding |

### 2. マッチングプロジェクト一覧

#### マッチカード構成

```
┌─────────────────────────────────────────┐
│ マッチ度: [██████████] 92%              │

│ [カバー画像サムネイル]                   │

│ プロジェクト名                           │
│ Phase: Light | カテゴリ: ものづくり      │
│                                         │

│ ・製造管理スキルが一致                   │

│                                         │
│ [詳細を見る]  [応募する]                 │
└─────────────────────────────────────────┘
```

#### マッチ度計算要素

| 要素 | 重み | 説明 |
|------|-----|------|
| スキルマッチ | 40% | 必要スキルとの一致度 |
| レベルマッチ | 25% | 必要レベルとの適合 |
| 稼働時間マッチ | 20% | 希望時間との適合 |
| 実績スコア | 15% | 過去の評価・実績 |

### 3. Boarding Track説明

| レベル | 条件 | 特典 |
|--------|------|------|
| Entry | 参画開始 | - |
| Candidate | 成果物1〜2本 + レビューA | 次フェーズ優先権 |
| Core | 2ヶ月継続 + KPI達成 | 報酬アップ |
| Boarding | 週Xh合意 + 役割確定 | 長期契約、将来的にSO検討 |

### 4. 参画中プロジェクト

| 表示項目 | 説明 |
|---------|------|
| プロジェクト名 | 参画中のPJ |
| 役割 | 担当役割 |
| Level | 現在のBoarding Level |
| ステータス | 進行中/完了 |
| 次の成果物 | 直近の成果物と期限 |
| ダッシュボードリンク | 詳細管理画面へ |
