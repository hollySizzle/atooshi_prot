# 06. ダッシュボード (DASHBOARD)

## 基本情報

| 項目 | 内容 |
|------|------|
| 画面ID | DASHBOARD |
| URL | `/dashboard` |
| 主要ユーザー | 全員（ロール別表示切替） |
| 目的 | 進捗管理、成果物管理、支払い管理 |

---

## 投資家への訴求ポイント

- **運用イメージ**の具体化
- **進捗の透明性**：検収→支払いの流れ
- **三者がつながる**プラットフォームとしての動き

---

## ロール別表示

| ロール | 主要表示内容 |
|--------|-------------|
| イノベーター | プロジェクト進捗、コミッター管理、CF状況 |
| コミッター | 担当成果物、報酬状況、Boarding進捗 |
| インベスター | 支援プロジェクト、リターン状況 |

※ モックでは**イノベーター視点**と**コミッター視点**を作成

## タブ詳細

### イノベーター視点

| タブ | 内容 |
|-----|------|
| 概要 | KPIサマリー、直近のアクション |
| クリティカルパス | 実行進捗のタイムライン表示 |
| チーム | コミッター一覧、応募者管理 |
| Bounty Pool | 支払い状況、履歴 |
| 支援者 | CF支援者一覧、メッセージ |

### コミッター視点

| タブ | 内容 |
|-----|------|
| 担当成果物 | 進行中/提出済み/完了 |
| 報酬・支払い | 確定報酬、支払い履歴 |
| Boarding進捗 | Level進捗、達成条件 |
